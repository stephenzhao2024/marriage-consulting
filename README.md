# marriage-consulting

给你的智能体加一个婚恋咨询大脑：不是一键算姻缘，而是先问清你的情况，再分析谁更适合你。

这是一个面向智能体的现实派婚恋咨询技能。它的工作方式不是直接抛结论，而是先补资料卡，再做定位、错位分析和匹配建议。

## 这个仓库包含什么

- 一个可直接使用的技能入口：`SKILL.md`
- 50 条 A 级公开视频 AI 文稿的匿名化索引
- 50 条匿名化原始文稿缓存
- 辅助处理脚本

## 它能做什么

- 择偶定位
- 条件错位分析
- 门当户对判断
- 高不成低不就纠偏
- “市场上找”还是“生活中找”的路径判断
- 现实派、短平快的婚恋咨询回答

## 它不做什么

- 没有图片或视频时，不做视觉颜值打分
- 不代替心理咨询
- 不代替最终配对裁决

## 回答方式

默认按这个顺序输出：

1. 先下一句话结论
2. 再指出最核心的错位点
3. 再判断更适合“市场上找”还是“生活中找”
4. 最后给 1 到 3 条行动建议

## 建议输入格式

把问题尽量补成一张资料卡：

- 年龄、城市、身高、体重
- 学历、职业、收入
- 家庭结构、父母资源、房车情况
- 恋爱史、婚史、子女情况
- 想找什么样的人

## 证据边界

这个公开版只使用 50 条同一公开咨询账号的 A 级抖音视频 AI 文稿。

- 保留了视频 ID、URL、文稿长度和 SHA256 去重记录
- 对标题、昵称和显式名称做了匿名化处理
- 不包含播客、外部评论、搜索摘要和标题级弱证据

## 适用场景

适合：

- 给已有智能体增加一个婚恋咨询视角
- 做文字版婚恋初筛和现实派分析
- 作为相亲/择偶问题的结构化回复模板

不适合：

- 当成真人替身
- 当成完整人物传记
- 当成视觉打分模型
